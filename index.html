1. Pilas (Stack)

Estructura LIFO. El último elemento agregado es el primero en ser eliminado.

  • Operaciones: PUSH (insertar) y POP (eliminar).
  • Uso típico: Gestión de llamadas a funciojnes.

2. Colas (Queue)

Estructura FIFO. El primer elemento que llega es el primero en ser eliminado.

  • Operaciones: Enqueue (agregar) y Dequeue (quitar).
  • Uso típico: Filas de impresión.

5. Árboles (Trees)

Estructura jerárquica con un nodo raíz y nodos hijos.

  • Tipos: Árbol Binario de Búsqueda (ABB).
  • Uso típico: Sistemas de archivos, índices de bases de datos.
